    Robo Analysts

     

    The trained models will be converted into "robo analysts."  So, each group will build a product.  The robo analysts will be uploaded to mgmt638.herokuapp.com.

    We conduct a large-scale empirical analysis, investigating nearly 30,000 individual stocks over 60 years from 1957 to 2016. 

     

    Our predictor set includes 94 characteristics for each stock, interactions of each characteristic with eight aggregate time-series variables, and 74 industry sector dummy variables, totaling more than 900 baseline signals.

     

    Some of our methods expand this predictor set much further by including nonlinear transformations and interactions of the baseline signals.

    Gu, Kelly, and Xiu

    Empirical Asset Pricing via Machine Learning

    Review of Financial Studies, 2022

    The economic gains from machine learning forecasts are large.

     

    We find that trees and neural networks unambiguously improve return prediction.

     

    When we form a long-short decile spread directly sorted on stock return predictions from a neural network, the strategy earns an annualized out-of-sample Sharpe ratio of 1.35 (value-weighted) and 2.45 (equal-weighted).

     

    Shallow learning outperforms deeper learning.

     

    The most successful predictors are price trends, liquidity, and volatility.

    Machine Learning Workflow

     

    1. Get data

     

    2. Define target variable

     

    Some possible targets for return prediction: 

    (a) return

    (b) return in excess of average return across stocks

    (c) rank of return among stocks

    (d) high, medium, low

     

     

    3. Choose predictors

     

    Some categories of predictors for return prediction:

     

    (a) financial ratios and changes in ratios

    (b) past returns, volatility, and volume

    (c) analyst forecasts, earnings surprises

    (d) dividend and earnings announcement calendars

    (e) insider trades, mutual fund trades, short interest

    (f) corporate announcements, news stories

    (g) social media sentiment

    (h) interactions with macroeconomic variables

    (i) industry membership

    4.  Choose a model

     

    Examples for return prediction

     

    (a) linear models

    (b) random and boosted forests

    (c) neural networks

     

    classification or regression?

    5. Choose transformations of predictors and target

     

     

    Turn categorical variables (industries) into dummy variables

     

    May want to include interactions (products) of variables

     

    May want to transform variables to avoid outliers

     

    May want to combine variables to reduce dimension

     

     

    6.  Decide on hyperparameters of models

     

     

    How much to constrain linear models

     

    How many trees and how many branches in forests

     

    How deep to make a neural network

    7.  Evaluate performance

     

    A standard criterion is mean squared error:

    average (prediction - actual)\(^2\)

     

    Sometimes better to use mean absolute error:

    average |prediction - actual|

     

    For stock return prediction, there may be better choices -

     

    If you predict a stock to earn 40% and buy it, is it an error if it actually earns 100%?

    Monthly data - predictors and returns - for largest 3,000 U.S. stocks each month from Jan, 2000 through Dec, 2021 (approximately 800,000 rows).

    Slide Classes in Editor UI

    You can use special syntax ("//= class-name") to specify custom class names that should be made available through the presentation editor interface. This makes it convenient to apply classes to specific slides. Here's an example class that blurs the slide that you apply it to:

    //= blurry-slide
.blurry-slide { filter: blur(5px); }

    Include fonts from Google or Typekit

    Click the T icon in the top right to load font families from Google Fonts or Typekit. Once you have added the fonts you want, don't forget to apply them to the content using CSS. For example, if you are loading the "Lobster" font family from Google Fonts you can apply it to all headings with: 

    .slides h1, .slides h2, .slides h3 {
  font-family: "Lobster";
}
